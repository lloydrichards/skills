# Practice route

Use when the learner must acquire a procedure, diagnostic habit, creative technique, or other productive skill.

## Sequence

1. State the observable performance and success criteria.
2. Calibrate guidance from current evidence.
3. Show a worked example with decisions explained at the point of use.
4. Give a faded case where the learner completes consequential steps.
5. Provide prompt explanatory feedback and optional ordered hints.
6. Give an independent, nearby transfer task without solution reveal.
7. Compare the result with the criteria and identify the next practice edge.

Use the worked-to-transfer primitive when HTML adds a useful feedback loop. For terminal, physical, or real-world performance, Markdown instructions plus conversational evidence may be better.

Do not count copying the worked example, completing trivial blanks, or following step-by-step instructions as independent demonstration.
