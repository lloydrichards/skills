# Lab route

Use when manipulating variables reveals a relationship that static exposition would obscure.

## Sequence

1. State the model, assumptions, variable roles, and observable outcome.
2. Ask the learner to predict a change before running or revealing it.
3. Let the learner manipulate only variables relevant to the target relationship.
4. Show the outcome in both visual and textual or tabular form.
5. Ask the learner to explain agreement or discrepancy.
6. Change the case and require a transfer prediction or diagnosis. Initialize it with `activityRole: "transfer"` so its evidence is distinguishable from guided work.
7. Export a learner-controlled evidence summary.

Use the prediction primitive as the activity spine. A simulation is an instrument, not evidence by itself.

Prefer native SVG or Canvas for small models. Add Plot, D3, p5, Motion, Matter.js, or Planck.js only when its capabilities materially simplify the truthful representation. Keep equations, approximations, numerical methods, and model limits visible.
