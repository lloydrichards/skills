# Pedagogy sources

These sources explain the research basis for `PEDAGOGY.md`. They are design provenance, not required runtime context.

- [Roediger & Karpicke, 2006, test-enhanced learning](https://pubmed.ncbi.nlm.nih.gov/16507066/) — retrieval can improve delayed retention even when repeated study produces greater confidence.
- [Cepeda et al., 2006, distributed-practice synthesis](https://pubmed.ncbi.nlm.nih.gov/16719566/) — distributed practice generally outperforms massed repetition, while useful timing varies.
- [Renkl et al., 2004, fading worked solution steps](https://eric.ed.gov/?id=EJ732331) — progressively completing solution steps can bridge worked examples and independent problem solving.
- [VanLehn et al., 2005, feedback meta-analysis](https://eric.ed.gov/?id=EJ1081708) — elaborated feedback is more useful than correctness-only responses for many higher-order outcomes.
- [Kalyuga, 2007, expertise reversal](https://pubmed.ncbi.nlm.nih.gov/21443379/) — guidance that helps novices can become redundant or harmful as expertise increases.

Treat these sources as design evidence, not universal prescriptions detached from domain, learner, or context.
