# Explain profile

**Reader job:** Build an accurate mental model.

**Reasoning chain:** Context → model → concrete example → mechanism → implications and limits.

Open with the one idea the reader should retain. Introduce only the background needed for that idea. Prefer an early concrete example, then connect implementation detail back to the model. End by stating what the model explains, where it stops applying, and useful next questions.

Avoid file-by-file narration, unexplained diagrams, and background that delays the central idea.
