# Audit profile

**Reader job:** Judge conformance or performance against stated criteria.

**Reasoning chain:** Objective and criteria → scope and method → observations → findings → conclusions → recommendations.

State the audit question and criteria before findings. Distinguish observations from evaluative conclusions. Explain sampling and evidence limitations. Recommendations must follow from findings and name the condition they would improve. End by answering the original audit question.

Avoid changing criteria after observing results, overstating sampled evidence, or making recommendations unrelated to findings.
