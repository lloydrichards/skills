# Learning profile

**Reader job:** Acquire and demonstrate one bounded capability or mental model.

**Reasoning chain:** Goal → prior model → new model → worked example → guided practice → feedback → transfer.

Open with a tangible learning outcome. Protect working memory by including only prerequisite knowledge required for the task. Use concrete examples before abstraction when possible. Practice should require retrieval or application and provide immediate explanatory feedback. End with a transfer prompt or next practice step.

Avoid passive exposition disguised as a lesson, trivia quizzes, and interaction that does not change understanding.
