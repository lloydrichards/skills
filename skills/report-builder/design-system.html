Typography and hierarchy

HTML heading ranks express document structure; type classes express visual role. Choose the correct element first, then apply the smallest class that communicates the intended emphasis. The specimens use neutral paragraphs to prove that no type role depends on a particular tag.

Figures and media

Color for data and status

Status colors communicate meaning and always appear with a text label. Data colors distinguish unrelated categories and follow a fixed sequence so neighboring values remain easier to tell apart.

Categorical data sequence

  • Category 1
  • Category 2
  • Category 3
  • Category 4
  • Category 5
  • Category 6

Use one data color by default. Add colors only when categories need differentiation, and pair them with direct labels, shapes, patterns, or separators.

Native controls

The shared baseline keeps native behavior while aligning controls with report surfaces, focus, accent, disabled, and invalid states. Every control still needs appropriate semantics, a persistent label, and any relevant hint or error association.
